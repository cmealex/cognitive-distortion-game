How to Play

Game Objective

The goal of Cognitive Distortion Matcher is to correctly match situations with the cognitive distortion type they represent.

Game Rules

  1. Each level contains a set of situations and cognitive distortions.
  2. Click on a situation on the left panel to select it.
  3. Then click on the matching cognitive distortion on the right panel.
  4. If correct, both will be marked with a checkmark (✓) and fade slightly.
  5. If incorrect, the distortion will flash red with an X mark (✗).
  6. Complete all matches before the timer runs out.
  7. Successfully completing a level unlocks the next level.

Level Progression

The game features 20 levels with increasing difficulty:

  • Levels 1-10: 3 matches per level
  • Levels 11-15: 4 matches per level
  • Levels 16-20: 5 matches per level
  • Higher levels feature more ambiguous situations and have tighter time limits.

Tips

  • Read each description carefully to understand the core characteristic of each distortion type.
  • Look for key phrases in the situations that hint at specific distortion types.
  • Practice identifying these patterns in your own thoughts during daily life.
  • Focus on speed at higher levels to beat the timer.
